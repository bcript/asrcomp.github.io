The Syllabus

The H2 Computing syllabus is designed by the Singapore Ministry of Education to develop computational thinking and programming skills. Students will gain a deep understanding of how computer systems work and how to create elegant solutions to complex problems.

Programming Fundamentals

  • Python programming language
  • Variables, data types, and operators
  • Control structures and functions
  • Object-oriented programming concepts

Data Structures & Algorithms

  • Arrays, lists, stacks, and queues
  • Sorting and searching algorithms
  • Algorithm efficiency and complexity
  • Recursion and dynamic programming

System Architecture

  • Computer organization and architecture
  • Operating systems fundamentals
  • Networks and the Internet
  • Database management systems

Software Development

  • Software development lifecycle
  • Requirements analysis and design
  • Testing and debugging strategies
  • Project management essentials

Assessment Structure

Paper 1 (40%): Written examination covering theoretical concepts, algorithm design, and problem-solving.

Paper 2 (30%): Practical programming examination where students solve problems using Python.

Coursework (30%): Individual project where students develop a complete software application addressing a real-world problem.
